SEC - FINGERSCAN SYSTEM

VB.NET POSTGRE SQL MICROSOFT ACCESS DIGITAL PERSONA BIOKEY FP VB6 MESSAGINGTOOLKIT.QRCODE VISUAL STUDIO 2010

When: 2021

Contribution: Full Stack Developer

Description: A Desktop Application developed in VB.NET and designed in Visual Studio 2010, PostgreSQL and Microsoft Access for its back-end. This Application is designed to be used for transacting Employee's Daily Time Record, Leave requests and other transactions.

SolidService Electronics Corporation Employees can now transact their daily time record on the Fingerscan System. They can log their IN/OUT Transaction using a Fingeprint Scanner to verify the fingerprint. They can Also file their Leave, Overtime, Official Business, Swap Day-Off And Log Validation on the system. Employees can also interact to the system to view announcements,birthdays and other concerns straight from the Head Office.

Fingerscan system is used accross 30+ SolidService Service Centers(Branches), Including the main office. Also this system is running in a Network designed environment, and can Transmit data from their respective Location into the Head Quarters (Main Office).

Other Modules

Leave Balance Inquiry

SEC Employee(s) can inquire how many leave balance credits they have. Data of Leave Balance Credits came from the Attendance System, then this system is transmitting the data to Fingerscan System across all branches in certain date.

Users can also view the detailed information on how many leave credits they already used, date which they used those leave credits, information on how many leave credits deductions based on their number of absents on a month, pending filed leave and others.

Viewing of Notice of Disciplinary Action

With the help of this module, HR Staff can now send a copy of the Notice of Disiciplinary Action (Memo) to Employees across branches through the Fingerscan System.

From the Attendance System, after generating the Notice of Disiciplinary Action based on accumulative lates and unexcused absences, it will send all the data to the Fingerscan System across all branches(Service Centers). Employee with Memo will be notified when they transact in the Fingerscan System.

Employee with Notice of Disciplinary Action will be required to acknowledge the generated memo on the Fingerscan System, User will be notified repeatedly until the acknowledgement will be done. User is given only limited time frame to acknowledge the Memo, or else his/her access on the Fingerscan System will be Disabled (Until the Memo Acknowlegement will be done). Based on the Sanction given to the employee, If they are sanctioned to be suspended, they Can't Transact on the Fingerscan System from the start till the end of their suspension.

Filing of Requests

Employees can file Overtime, Official Business, Leave of Absence, LOG Validation and Swap Day-Off on the Fingerscan System. Once Filed, they will be given a reference number whick they can track the status of the request.

When Filing Leave of Absence Request there is some algorithm i added that they cannot file leave of request if the employee don't have enough Leave Credits (Based on how many Days/Hours the duration of their Leave of Absence).
